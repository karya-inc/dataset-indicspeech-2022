Audio format
============

Encoding: PCM
Bitwidth: 16
Sampling Frequency: 16 KHz

Folder structure
================

Folder: wav

Contains all the audio recordings. Each name has two fields separated by a '-'. First field is speaker ID and the second field is the sentence ID.

File: transriptions.txt

This file contains the transcriptions for all audio recordings. Each line has two fields separated by a tab. First field contains the name of the recording file (without the extension). The second field contains the transcription. All transcriptions are in the dominant script of the respective language.
